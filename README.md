# gpu-pt
A very simple gpu side path tracer for learning purposes.

## Features
- Rendered on the gpu using wgpu
- Obj file loading
- Hdri environment
- Bvh acceleration
- Anti-aliasing
- Easily expandable materials
- Simple color management
