# gpu-pt
A simple gpu side path tracer for learning purposes.

## Features
- Rendered on the gpu using wgpu
- Obj file loading
- Hdri environment
- Bvh acceleration
- Anti-aliasing
- Easily expandable materials
- Basic color management
